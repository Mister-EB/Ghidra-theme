## 新增主題
**1. 從此儲存庫下載您想要的主題**

**2. 在 Ghidra 初始頁面中，點擊螢幕右上角的 Edit**

**3. 點擊 Theme 上並點擊 Import...**

**4. 找到並選擇您之前下載的 .theme 文件**  

**<p align="center">🎉🎉🎉這樣就能夠成功新增主題了！🎉🎉🎉</p>**
